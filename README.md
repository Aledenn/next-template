## 这是一个nextjs的项目模版

#### quick start

```
  yarn
  yarn run dev
```
build and deploy
```
  yarn run build
```

支持以下特性 feature:
  1. less module --- *.module.less使用css module *.less不使用css module
  2. bundle analizyer 打包分析 
  3. i18n 内置国际化
  4. eslint

后续：
  redux支持
